# 步骤六：设置界面

## 目标

在步骤五已经完成的游戏封面基础上，新增设置界面。

玩家在封面界面点击 `fmstart` 后，弹出设置界面。

设置界面用于设置游戏人数、游戏速度和游戏时间。

## 基础约束

1. 不破坏步骤一到步骤五已经确认的游戏主流程和封面显示规则。
2. 游戏仍然使用 Cocos Creator 3.8.8。
3. 画布仍然为横屏 1280x720。
4. 设置界面显示在封面和游戏画面之上。
5. 设置界面显示时，下方封面或游戏内容不能被点击。
6. 设置界面关闭后，不应残留遮挡层。
7. Cocos Creator 控制台不能出现 missing script 或脚本挂载错误。
8. `main.scene` JSON、资源 UUID、脚本字段挂载关系需要通过校验。

## 设置参数

设置界面维护 3 个参数：

1. `gameperson`
   可选值：`single`、`double`
   显示含义：单人、双人
   默认值：`single`

2. `gamespeed`
   可选值：`normal`、`fast`
   显示含义：正常、快速
   默认值：`normal`

3. `gametime`
   可选值：1-9
   显示含义：游戏时间分钟数
   默认值：3

## 设置界面对象

设置界面包含以下对象：

1. `setmask`
   用途：半透明暗色遮罩，用于让背景变暗并阻止下方内容点击。
   实现说明：使用全屏 Sprite 节点，尺寸 1280x720，显示在设置面板下方。

2. `setpanel`
   用途：设置面板底图。
   资源：`CC-003/assets/art/菜单/cd12.png`
   说明：`cd12.png` 已经包含设置界面的左侧图标文字、时间设置区的浅色长条背景和中间数字选中框。

3. `setdouble`
   用途：双人按钮。
   选中资源：`CC-003/assets/art/菜单/cd4.png`
   未选中资源：`CC-003/assets/art/菜单/cd8.png`

4. `setsingle`
   用途：单人按钮。
   选中资源：`CC-003/assets/art/菜单/cd10.png`
   未选中资源：`CC-003/assets/art/菜单/cd5.png`

5. `setfast`
   用途：快速按钮。
   选中资源：`CC-003/assets/art/菜单/cd7.png`
   未选中资源：`CC-003/assets/art/菜单/cd11.png`

6. `setnormal`
   用途：正常按钮。
   选中资源：`CC-003/assets/art/菜单/cd9.png`
   未选中资源：`CC-003/assets/art/菜单/cd6.png`

7. `setclose`
   用途：关闭按钮。
   资源：`CC-003/assets/art/封面/close.png`

8. `setok`
   用途：确认按钮。
   资源：`CC-003/assets/art/菜单/cd1.png`
   说明：原始表达第 8 条中的 `setconfirm` 视为 `setok` 的同义描述，正式对象名使用 `setok`。

9. `settime`
   用途：游戏时间数字转盘容器。
   位置：参考图二中显示 3、4、5 数字的横向时间选择区域。
   规则：显示 3 个数字，中间数字为当前选中时间。
   默认显示：2、3、4。
   默认选中：3。
   显示说明：时间设置区的浅色长条背景和中间选中外框由 `setpanel` 的 `cd12.png` 自带，`settime` 不额外显示背景图。

10. `settimeLeft`
    用途：时间转盘左侧数字显示。
    显示说明：只显示数字，不显示独立底框。

11. `settimeCenter`
    用途：时间转盘中间数字显示，也是当前选中数字。
    显示说明：只显示数字，数字位于 `cd12.png` 自带的中间选中框上方。

12. `settimeRight`
    用途：时间转盘右侧数字显示。
    显示说明：只显示数字，不显示独立底框。

13. `settimeTouchNodes`
    用途：时间转盘触摸区域集合。
    包含：`settime`、`settimeLeft`、`settimeCenter`、`settimeRight`。
    说明：用于保证玩家在时间数字区域左右滑动时稳定触发时间切换。

## 显示规则

1. 游戏进入后仍先显示封面。
2. 设置界面默认不显示。
3. 点击封面中的 `fmstart` 后，显示设置界面。
4. 设置界面显示时，`setmask` 全屏显示，背景变暗。
5. 设置界面显示时，`setpanel` 居中显示。
6. 设置界面显示时，设置界面对象显示在封面对象之上。
7. 设置界面显示时，下方封面对象不能响应点击。
8. 设置界面关闭后，所有设置界面对象隐藏。
9. 设置界面关闭后，封面继续显示。
10. 本步骤中点击 `setok` 保存设置、关闭设置界面，并开始游戏。

## 按钮分组规则

### 游戏人数

1. `setdouble` 和 `setsingle` 是一组。
2. 该组任意时刻必须有且只有一个按钮处于选中状态。
3. 默认选中 `setsingle`。
4. 点击 `setdouble` 后，`gameperson = double`。
5. 点击 `setsingle` 后，`gameperson = single`。

### 游戏速度

1. `setfast` 和 `setnormal` 是一组。
2. 该组任意时刻必须有且只有一个按钮处于选中状态。
3. 默认选中 `setnormal`。
4. 点击 `setfast` 后，`gamespeed = fast`。
5. 点击 `setnormal` 后，`gamespeed = normal`。

## 时间转盘规则

1. `settime` 是游戏时间选择控件容器。
2. `gametime` 可选值为 1-9。
3. 默认 `gametime = 3`。
4. 时间转盘同一时间显示 3 个数字。
5. 中间数字为当前选中的 `gametime`。
6. 默认显示 2、3、4。
7. 数字循环显示，1 的左侧是 9，9 的右侧是 1。
8. 向左滑动 `settime` 时，当前选中数字增加 1。
9. 向右滑动 `settime` 时，当前选中数字减少 1。
10. 滑动结束后，中间区域停留的数字为当前设置时间。
11. 拨动过程中，`settimeLeft`、`settimeCenter`、`settimeRight` 三个数字需要跟随手指左右移动。
12. 拨动速度越快，数字横向移动越快。
13. 拨动距离超过一个数字间距时，需要立即循环切换数字。
14. 松开手后，时间拨盘必须吸附回完整数字位置。
15. 松开手后，不能出现半个数字停留在显示区内。
16. 拨动过程中，数字不能移动到 `cd12.png` 时间显示区域之外。
17. 数字不能移出时间显示区域的实现方式，应优先使用 `settime` 作为裁剪容器限制可见范围。
18. 不能通过硬性截断数字横向位移来限制显示范围，否则数字进入中间区域时会出现卡顿或突然跳变。
19. 拨动过程中，数字的横向位置必须连续跟随手指移动。
20. 时间设置区的浅色长条背景由 `CC-003/assets/art/菜单/cd12.png` 自带，不额外叠加独立底板。
21. 中间选中区域外框由 `CC-003/assets/art/菜单/cd12.png` 自带，不额外叠加 `cd3.png`。
22. `settimeLeft` 和 `settimeRight` 只显示数字，不能出现独立外框。
23. `settimeCenter` 只显示数字，数字显示在 `cd12.png` 自带的中间选中框上方。
24. 数字显示使用 `CC-003/assets/art/菜单/szz.png` 作为数字资源依据。
25. 实现时如不切割 `szz.png`，允许先使用 Cocos Label 显示 1-9 数字，但需要保留 `szz.png` 作为该控件的数字资源依据。

## 关闭和确认规则

1. 点击 `setclose`，关闭设置界面，不保存本次临时修改。
2. 点击 `setok`，保存当前设置参数，关闭设置界面，并开始游戏。
3. 打开设置界面时，需要从已保存设置生成临时设置。
4. 点击设置按钮或滑动时间转盘时，只修改临时设置。
5. 点击 `setclose` 时，丢弃临时设置，恢复已保存设置显示。
6. 点击 `setok` 时，将临时设置写入已保存设置。
7. 点击 `setok` 后，主游戏流程从步骤一的开场掉落阶段开始。
8. 已保存设置后续用于影响游戏逻辑。

## 入口规则

1. 步骤六覆盖步骤五中的封面入口规则。
2. 点击 `fmlab01` 不开始游戏。
3. 点击 `fmlab02` 不开始游戏。
4. 点击 `fmclose` 不开始游戏。
5. `fmstart` 作为设置界面入口。
6. 点击 `fmstart` 时只弹出设置界面，不直接开始游戏。
7. 点击 `setclose` 后关闭设置界面并返回封面。
8. 点击 `setok` 后保存设置、关闭设置界面，并开始游戏。

## 验收标准

1. 场景中存在设置界面节点和相关按钮节点。
2. 设置界面默认隐藏。
3. 封面显示时，点击 `fmstart` 后设置界面显示。
4. 设置界面显示时背景变暗。
5. 设置界面显示时下方封面不能被点击。
6. 默认 `gameperson = single`，`setsingle` 显示选中状态，`setdouble` 显示未选中状态。
7. 点击 `setdouble` 后，`setdouble` 选中，`setsingle` 未选中。
8. 点击 `setsingle` 后，`setsingle` 选中，`setdouble` 未选中。
9. 默认 `gamespeed = normal`，`setnormal` 显示选中状态，`setfast` 显示未选中状态。
10. 点击 `setfast` 后，`setfast` 选中，`setnormal` 未选中。
11. 点击 `setnormal` 后，`setnormal` 选中，`setfast` 未选中。
12. 默认 `gametime = 3`，时间转盘显示 2、3、4。
13. 向左滑动时间转盘后，选中时间增加。
14. 向右滑动时间转盘后，选中时间减少。
15. 时间转盘数字在 1-9 内循环。
16. 拖动时间转盘时，三个数字需要随手指左右移动。
17. 拨动速度越快，数字横向移动越快。
18. 松开手后，时间转盘必须吸附回完整数字显示位置。
19. 松开手后，不能出现半个数字。
20. 拖动过程中，数字不能移出时间显示区域。
21. 拖动过程中，数字必须连续跟手移动，不能在进入中间选中区域时出现明显卡顿。
22. 时间转盘的浅色长条和中间选中框来自 `cd12.png`，不能额外叠加造成重复边框。
23. `settimeLeft` 和 `settimeRight` 周围不能出现独立外框。
24. 只有中间选中数字位于 `cd12.png` 自带的中间外框区域内。
25. 点击 `fmlab01` 不开始游戏。
26. 点击 `fmlab02` 不开始游戏。
27. 点击 `fmclose` 不开始游戏。
28. 点击 `setclose` 后，设置界面关闭，并且不保存本次临时修改。
29. 点击 `setok` 后，设置界面关闭，保存当前设置，并开始游戏。
30. 设置界面关闭后，封面仍可继续操作。
30. Cocos Creator 控制台没有 missing script 或脚本挂载错误。
31. `main.scene` JSON 解析通过。
32. `main.scene` 中新增字段与 `GameStart.ts` 中 `@property` 字段一致。
